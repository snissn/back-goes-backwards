# Axial Load Gradient Model

A mechanics-first model describing how load routes through the body and how healthy movement tends to shed load downward rather than accumulate it upward.

This is not medical advice and does not diagnose or treat disease. Stop and seek medical evaluation for red flags such as progressive neurologic deficits, bowel or bladder changes, fever, severe unremitting pain, or recent significant trauma.

## How to use this site

- Read the model definition first.
- Run the Quick screen in under 2 minutes.
- Use the drill taxonomy to pick a minimal protocol.
