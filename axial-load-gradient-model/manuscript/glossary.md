# Glossary

- **Axial coordinate (z)**: a normalized vertical coordinate from ground to head.
- **Load function (L)**: net mechanical load transmitted through the axial system.
- **Axial load gradient (G)**: `dL/dz`, the rate of load change along the axis.
- **Piecewise gradients (g_L, g_T, g_C)**: segment-averaged gradients for lumbar, thoracic, cervical.
- **Inversion locus**: a region where the gradient changes sign.
